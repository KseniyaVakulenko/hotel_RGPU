# План проекта «Гостиница РГПУ им. А. И. Герцена»

**Автор:** (ваше имя)

## 1. Главная цель (SMART)

Разработать веб-приложение для онлайн-бронирования номеров гостиницы РГПУ
в течение 3 месяцев. Приложение должно позволять гостям просматривать свободные
номера и бронировать их без звонка администратору.

**SMART-критерии:**
- **S (Конкретность):** сайт для бронирования номеров гостиницы РГПУ
- **M (Измеримость):** минимум 3 страницы (каталог, личный кабинет, админ-панель)
- **A (Достижимость):** используем React + готовые библиотеки
- **R (Значимость):** автоматизирует работу администратора гостиницы
- **T (Ограничение по времени):** 3 месяца

## 2. Предыстория

Сейчас бронирование номеров происходит по телефону или email.
Администратор вручную ведёт учёт в Excel. Это неудобно и долго.
Нужен сайт, который покажет номера онлайн и позволит бронировать без участия человека.

## 3. Границы проекта

### Технические границы:
- Фронтенд: React + TypeScript
- Бэкенд: имитация API (заглушка)
- Хранение данных: временно в памяти браузера (localStorage)

### Функциональные границы:
- Просмотр каталога номеров
- Фильтрация по датам
- Форма бронирования
- Личный кабинет гостя
- Панель администратора

### Что НЕ входит в проект:
- Онлайн-оплата
- Мобильное приложение
- Интеграция с 1С

## 4. Декомпозиция работ (WBS)

1. **Подготовка**
   - Настройка ESLint, Prettier, Husky
   - Настройка Storybook и Docusaurus

2. **Дизайн**
   - Разработка макетов страниц
   - Создание UI-компонентов

3. **Разработка**
   - Верстка страниц
   - Реализация логики бронирования
   - Создание личного кабинета

4. **Документирование**
   - Написание требований
   - Создание диаграмм
   - Описание API
