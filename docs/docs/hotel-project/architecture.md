# Архитектура системы

**Автор:** (имя третьего члена команды)

## 1. Общая архитектура

Система построена по принципу **клиент-сервер**.

- **Клиент (Frontend):** React-приложение, работает в браузере пользователя.
  Отображает интерфейс и отправляет запросы к серверу.
- **Сервер (Backend):** обрабатывает запросы, работает с базой данных.
  Для учебного проекта используется заглушка API.

## 5. Описание API

### GET /api/rooms

Получение списка номеров.

**Параметры запроса:**
- `checkIn` (string, формат ГГГГ-ММ-ДД) — дата заезда
- `checkOut` (string, формат ГГГГ-ММ-ДД) — дата выезда
- `guests` (number) — количество гостей

**Пример запроса:**
```
GET /api/rooms?checkIn=2026-05-01&checkOut=2026-05-05&guests=2
```

**Пример ответа (200 OK):**
```json
{
  "rooms": [
    {
      "id": 1,
      "title": "Одноместный Стандарт",
      "price": 2500,
      "capacity": 1,
      "imageUrl": "https://example.com/room1.jpg",
      "isAvailable": true
    }
  ]
}
```

### POST /api/bookings

Создание нового бронирования.

**Заголовки:**
- `Content-Type: application/json`
- `Authorization: Bearer {token}`

**Тело запроса:**
```json
{
  "roomId": 1,
  "checkIn": "2026-05-01",
  "checkOut": "2026-05-05",
  "guestsCount": 2,
  "guestName": "Иванов И.И.",
  "guestEmail": "ivanov@example.com",
  "guestPhone": "+7-999-123-45-67"
}
```

**Пример ответа (201 Created):**
```json
{
  "bookingId": 123,
  "status": "Ожидает подтверждения",
  "roomTitle": "Одноместный Стандарт",
  "totalPrice": 10000,
  "createdAt": "2026-04-24T15:30:00Z"
}
```

## 6. Нефункциональные требования

| Требование | Описание |
|:---|:---|
| **Безопасность** | Авторизация по JWT-токену. Пароли хешируются |
| **Производительность** | Время ответа API — не более 500 мс |
| **Доступность** | Сайт доступен 99% времени в рабочее время (8:00–22:00) |
| **Поддержка браузеров** | Chrome, Firefox, Safari, Edge последних версий |
| **Язык** | Русский |
| **Хранение данных** | Данные о бронированиях хранятся 5 лет |
| **Масштабируемость** | Архитектура позволяет добавить новые типы номеров |
